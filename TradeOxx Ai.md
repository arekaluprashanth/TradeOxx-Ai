# TradeOxx Ai - Project Architecture & Overview

Welcome to the TradeOxx Ai source code! This file is designed to make the whole project easily **understandable** for any developer.

## Tech Stack
- **Frontend Framework:** React 18 with TypeScript
- **Routing:** React Router v6
- **Styling:** Tailwind CSS (Custom dark theme with glassmorphism and drop-shadow glows)
- **State Management:** Zustand (`src/store/marketStore.ts`)
- **Animations:** Framer Motion (Global hover scale pop-ups)
- **Mobile Wrapper:** Capacitor JS (Android generation)
- **Build Tool:** Vite

## Directory Structure

### `/client/src/`
This is the core of the application where all the React code lives.

- **`/components`** - Reusable UI building blocks.
  - `/auth` - Login, Signup, and Authentication pages.
  - `/dashboard` - Core dashboard layout, TopNav, BottomNav, Sidebar.
  - `/charts` - Trading charts, order books, and depth charts (using Recharts).
  - `/portfolio` - User portfolio overview, asset allocation pie charts, trade history.
  - `/strategy` - Strategy builder, AI backtesting results, and automated trading logic.
  - `/ui` - Generic UI components (Buttons, Cards, Modals, ChatBot).
- **`/pages`** - The main page views that map to routes (Dashboard, Markets, Portfolio, Strategy, Watchlist, Profile).
- **`/store`** - Zustand state management files handling live mock data for cryptocurrencies and user balances.

### `/android/`
This directory contains the native Android Studio project generated by Capacitor. 
- It wraps the `/client/dist/` web build into a native Android WebView.
- The compiled APK is outputted to `android/app/build/outputs/apk/release/TradeOxx Ai.apk`.

## Key Features & Design Choices
1. **165Hz Optimized Animations:** All animations (like the floating crypto logos and global hover effects) use hardware-accelerated CSS transforms (`scale`, `translate`) rather than heavy DOM repaints to ensure maximum smoothness on high-refresh-rate screens.
2. **Global Pop-up Effect:** Every interactive element globally scales up by 1.05x on hover for tactile feedback without relying on harsh background colors.
3. **Glassmorphism:** The UI uses heavily translucent backgrounds (`bg-dark-800/30`, `backdrop-blur`) combined with vibrant Cyan (`#00D09C`) and neon glowing drop-shadows to create a premium Fintech aesthetic similar to advanced trading terminals.

## How to Run Locally
1. Open a terminal in the `client` directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.

## How to Build the Android APK
1. Run `npm run build` inside the `client` directory.
2. Run `npx cap sync android` to sync the web build with the Android project.
3. Navigate to the `android` directory and run `gradlew assembleRelease` to compile the final `.apk`.
